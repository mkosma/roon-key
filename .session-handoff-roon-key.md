## Architecture notes

Single Node process on mini, one Roon Core WebSocket via `node-roon-api`, two surfaces:
- `POST /mcp` (Streamable HTTP) - MCP tools registered in `tools/zone.ts`, `tools/playback.ts`, etc.
- `/control/*` + `/config/*` REST - in `control/control-router.ts`, for roon-key + iOS Shortcuts

All routes through `roonConnection` singleton in `src/roon-connection.ts`. Auth: `BRIDGE_AUTH_TOKEN` env var, Bearer on `/mcp`, `/control`, `/config`. `/health` is open. mDNS advertises `_roon-bridge._tcp.local` on `BRIDGE_PORT=3100`.
